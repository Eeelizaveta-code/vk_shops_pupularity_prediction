## Модель, предсказывающая количество просмотров, лайков и покупок с клипа-рекламы VK у предполагаемой комбинации автор+товар

Состоит из 
- детального анализа данных по таблицам Vk Shops;
- расчета ключевых метрик;
- полной стратегии по развитию платформы:
  - составлению каталога,
  - взаимодействию и с селлерами, и с авторами контента, и с покупателями;
- составления грамотного фичсета для модели по рекомендации авторам потенциально прибыльных товаров;
- разметки фичсета;
- обучения модели;
- интерпретации результатов;
- дизайна варфреймов по потенциальному приминению результатов в ЛК пользователя.

Для более подробного ознакомления рекомендована презентация .pdf, для совсем подробного - основной файл .ipynb.
